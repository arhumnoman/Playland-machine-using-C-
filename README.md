# Playland Boxer

A new C# starter project for the Playland Boxer application.

## Structure

- `PlaylandBoxer.csproj` - .NET 8 console application project file
- `Program.cs` - application entry point
- `Game.cs` - basic game runner placeholder
- `GameAssets.cs` - placeholder asset loader with `Assets` folder support
- `Assets/` - place your graphics and video files here

## Next steps

1. Add your graphics and video assets to `PlaylandBoxer/Assets`
2. Update `Game.cs` and `GameAssets.cs` to load and render your content using your preferred graphics library
3. Run the project with `dotnet run --project PlaylandBoxer/PlaylandBoxer.csproj`
